kobo2readwise

Sync highlights from sideloaded Kobo books to Readwise.

This browser isn't supported. kobo2readwise reads your Kobo database locally using the File System Access API, which today only works in Chrome or Edge on desktop. Please reopen this page there. Firefox, Safari and mobile browsers can't run it.

Highlights from books you sideloaded onto your Kobo (via Calibre, email or USB) live only in the device's local KoboReader.sqlite — Readwise can't fetch them automatically. This tool reads that file in your browser and forwards the highlights to Readwise through a thin proxy that never stores or logs your token. Store, Kobo Plus and library books already sync via Readwise's own integration.

1 Open your Kobo database

Plug in your Kobo, then pick KoboReader.sqlite from the hidden .kobo folder on the device. The file is read locally and never uploaded.
