About & Help
⚠️

This browser can't run kobo2readwise

kobo2readwise reads your Kobo database directly on your computer using the File System Access API. That API only exists in a couple of desktop browsers.

✅ Chrome — desktop ✅ Edge — desktop

Firefox, Safari, and every phone or tablet browser can't run it — there's no workaround on those.

Open this page in Chrome or Edge on a computer to continue.
